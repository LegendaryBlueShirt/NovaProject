# NovaProject
A frontend for running One Must Fall 2097 in DOSBox

## Prerequisites
- A mingw installation (Windows)

- SDL2.0

- SDL_Image

- SDL_TTF

## Building
### Windows
Change nativelibs in build.gradle.kts to the appropriate mingw location.  Run the gradle task
```bash
gradlew runWithRes
```
